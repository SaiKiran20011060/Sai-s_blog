What is Regression?

Regression is a statistical method used to model the relationship between a dependent variable (target) and one or more independent variables (features). It's used to predict continuous numerical values.

Key Goal: Find the best line/curve that fits through data points to make predictions.

Types of Regression Problems:

  • Simple Linear: One feature predicts one target
  • Multiple Linear: Multiple features predict one target
  • Polynomial: Non-linear relationships using polynomial terms
  • Support Vector: Uses support vectors and kernels
  • Tree-based: Uses decision trees (Decision Tree, Random Forest)

Common Applications:

  • House price prediction
  • Stock market forecasting
  • Sales revenue estimation
  • Temperature prediction
  • Medical diagnosis scoring

A form of regression that models non-linear relationships by using polynomial terms of the input features.

рџЊі Decision Tree Regression

Uses a tree-like model of decisions to predict continuous values by splitting data based on feature values.

How it Works:

1. Start with entire dataset 2. Find best feature and split point that minimizes MSE 3. Split data into two branches 4. Repeat recursively for each branch 5. Stop when criteria met (max depth, min samples, etc.)

Split Criteria (MSE):

MSE = (1/n) Г— ОЈ(yбµў - Е·)ВІ

Best split minimizes weighted MSE: MSE_split = (n_left/n) Г— MSE_left + (n_right/n) Г— MSE_right

Prediction:

For each leaf node: Е· = mean(y_values_in_leaf)

Root: All houses (avg price: $300k) в”њв”Ђв”Ђ Size в‰¤ 1500 sq ft (avg: $250k) в”‚ в”њв”Ђв”Ђ Bedrooms в‰¤ 2 в†’ $200k в”‚ в””в”Ђв”Ђ Bedrooms > 2 в†’ $300k в””в”Ђв”Ђ Size > 1500 sq ft (avg: $400k) в”њв”Ђв”Ђ Age в‰¤ 10 years в†’ $450k в””в”Ђв”Ђ Age > 10 years в†’ $350k New house: 1800 sq ft, 3 bedrooms, 5 years old Path: Root в†’ Size > 1500 в†’ Age в‰¤ 10 в†’ Prediction: $450k
Hyperparameters: max_depth, min_samples_split, min_samples_leaf, max_features

рџЊІ Random Forest Regression

An ensemble method that combines multiple decision trees to create a more robust and accurate model.

Algorithm:

1. Create n bootstrap samples from training data 2. For each sample, train a decision tree with: вЂў Random subset of features at each split вЂў Different random seed 3. For prediction: average all tree predictions

Е· = (1/n) Г— ОЈ(tree_i_prediction)

Key Concepts:

  • Bootstrap Sampling: Random sampling with replacement
  • Feature Randomness: Only consider subset of features per split
  • Bagging: Bootstrap Aggregating reduces overfitting
  • Out-of-Bag (OOB): Use unsampled data for validation

Example: Ensemble Prediction

House: 1600 sq ft, 3 bedrooms, 8 years old Tree 1 prediction: $320k (trained on sample 1) Tree 2 prediction: $340k (trained on sample 2) Tree 3 prediction: $310k (trained on sample 3) Tree 4 prediction: $330k (trained on sample 4) Tree 5 prediction: $325k (trained on sample 5) Random Forest prediction: (320 + 340 + 310 + 330 + 325) / 5 = $325k

Feature Importance:

Importance = ОЈ(weighted_impurity_decrease) / n_trees

Higher values indicate more important features

рџ“Џ R-Squared (Coefficient of Determination)

R-squared measures how well the regression model explains the variability in the target variable. It's the most common metric for evaluating regression models.

Formula:

RВІ = 1 - (SS_res / SS_tot)

Where:
вЂў SS_res = ОЈ(yбµў - Е·бµў)ВІ (Residual Sum of Squares)
вЂў SS_tot = ОЈ(yбµў - Иі)ВІ (Total Sum of Squares)
вЂў yбµў = Actual values
вЂў Е·бµў = Predicted values
вЂў Иі = Mean of actual values

Alternative Formula:

RВІ = (SS_reg / SS_tot)

Where SS_reg = ОЈ(Е·бµў - Иі)ВІ (Regression Sum of Squares)

Note: SS_tot = SS_reg + SS_res

Interpretation:

  • RВІ = 1.0: Perfect fit (model explains 100% of variance)
  • RВІ = 0.8: Good fit (model explains 80% of variance)
  • RВІ = 0.5: Moderate fit (model explains 50% of variance)
  • RВІ = 0.0: Model performs as well as predicting the mean
  • RВІ < 0: Model performs worse than predicting the mean

Example Calculation:

Actual values: [100, 120, 140, 160, 180] Predicted values: [110, 125, 135, 155, 175] Mean of actual: 140 SS_res = (100-110)ВІ + (120-125)ВІ + (140-135)ВІ + (160-155)ВІ + (180-175)ВІ = 100 + 25 + 25 + 25 + 25 = 200 SS_tot = (100-140)ВІ + (120-140)ВІ + (140-140)ВІ + (160-140)ВІ + (180-140)ВІ = 1600 + 400 + 0 + 400 + 1600 = 4000 RВІ = 1 - (200/4000) = 1 - 0.05 = 0.95 Interpretation: Model explains 95% of the variance

Adjusted R-Squared:

Adjusted RВІ = 1 - [(1-RВІ)(n-1)/(n-k-1)]

Where:
вЂў n = Number of observations
вЂў k = Number of features

Penalizes adding irrelevant features
When to Use:
вЂў Comparing different regression models
вЂў Understanding model performance
вЂў Feature selection (higher RВІ with fewer features is better)
вЂў Model validation and improvement
